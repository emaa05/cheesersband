## 🔨 Funcionalidades 

### Registro de Usuarios 📝
La aplicación ofrece un formulario intuitivo que permite a los usuarios registrarse fácilmente. Los usuarios pueden ingresar la información necesaria y crear sus cuentas de manera rápida.

### Inicio de Sesión Seguro 🔐
Para garantizar la seguridad, hemos implementado un formulario de inicio de sesión. El sistema utiliza la tecnología JWT (JSON Web Tokens) para proporcionar a cada usuario un token único. Este token se utiliza para autenticar las solicitudes del usuario y mantener un entorno seguro.

### Cierre de Sesión 🚪
Los usuarios pueden cerrar sesión de manera segura en cualquier momento, lo que invalida el token actual y garantiza la privacidad de su cuenta.

### Compra de Tickets 🎟️
La aplicación permite a los usuarios comprar tickets para eventos o servicios específicos. Implementamos un sistema de compra seguro para facilitar la experiencia del usuario.

### Gestión de Álbumes y Canciones 🎵

**Agregar Álbumes y Canciones**: Los usuarios pueden añadir nuevos álbumes a su colección, cada uno con su lista de canciones correspondiente.
  
**Editar Álbumes y Canciones**: La aplicación proporciona la capacidad de editar información detallada sobre álbumes y canciones existentes, brindando flexibilidad y personalización.

**Eliminar Álbumes**: Los usuarios tienen la opción de eliminar álbumes completos, eliminando así todas las canciones asociadas.


## 🔧 Instalación 

Paso a paso de cómo descargar el repositorio y levantarlo. Ejemplo: 

1- Clonar el repositorio.

2- Instalar dependencias corriendo el comando _npm i_.

3- Correr el proyecto con : _npm start_.
